# دليل تشغيل المشروع

مرحباً بك في مشروع IS424! هذا الدليل مخصص ويشرح كيفية تشغيل المشروع على جهازك خطوة بخطوة.

## المتطلبات الأساسية

1. **تثبيت بايثون**:
   - يجب أن يكون لديك بايثون مثبت على جهازك (يفضل الإصدار 3.8 أو أحدث).
   - يمكنك تحميل بايثون من الموقع الرسمي: https://www.python.org/downloads/

2. **تثبيت pip**:
   - غالباً يتم تثبيت pip تلقائياً مع بايثون. إذا لم يكن مثبتاً، يمكنك البحث عن طريقة تثبيته حسب نظام التشغيل الخاص بك.

3. **تثبيت Django**:
   - افتح نافذة الأوامر (Terminal) واكتب:
     ```
     pip install django
     ```


## خطوات تشغيل المشروع

1. **تحميل ملفات المشروع**:
    - قم بتحميل ملفات المشروع إلى جهازك (عن طريق رابط التحميل أو نسخة من المعلم).

2. **الدخول إلى مجلد المشروع**:
    - افتح نافذة الأوامر (Terminal) وانتقل إلى مجلد المشروع باستخدام الأمر:
       ```
       cd المسار/إلى/مجلد/المشروع
       ```
    - مثال:
       ```
       cd Desktop/is424_project
       ```

3. **تشغيل أوامر التهيئة (migrate)**:
    - قبل تشغيل السيرفر، يجب تنفيذ أوامر التهيئة لقاعدة البيانات:
       ```
       python manage.py migrate
       ```
    - هذا الأمر يُجهز قاعدة البيانات للاستخدام.

4. **تشغيل الخادم المحلي (السيرفر)**:
    - اكتب الأمر التالي في نافذة الأوامر:
       ```
       python manage.py runserver
       ```
    - إذا ظهرت رسالة أن السيرفر يعمل، انتقل للخطوة التالية.

5. **فتح الموقع في المتصفح**:
    - افتح متصفح الإنترنت واكتب في شريط العنوان:
       ```
       http://127.0.0.1:8000/
       ```
    - ستظهر لك الصفحة الرئيسية للمشروع.

## ملاحظات مهمة
- إذا واجهت أي مشكلة، تأكد من أنك في المجلد الصحيح وأن بايثون وDjango مثبتان بشكل صحيح.
- إذا ظهرت رسالة خطأ، انسخها وابحث عنها في الإنترنت أو اسأل المعلم.
